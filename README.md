# CareerFlow

一个基于 React、TypeScript、Vite 和 Tailwind CSS 的日本就活管理工具。

## 功能

- 添加、编辑、删除任务
- 勾选任务完成状态
- LocalStorage 自动保存任务与深色模式偏好
- 全部 / 进行中 / 已完成筛选与清除已完成
- 深色模式
- 手机和桌面端响应式布局

## 启动

需要 Node.js 20 或更高版本。

```bash
npm install
npm run dev
```

终端会显示本地访问地址，通常是 `http://localhost:5173`。

## 生产构建

```bash
npm run build
npm run preview
```
